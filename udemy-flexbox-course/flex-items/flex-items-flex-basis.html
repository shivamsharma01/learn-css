If We set Width = 60px; and then change flex-direction from row to column on a mobile device, width would still be 60px. But actually we wanted the flex-item to cover the entire row. Using Flex Basis, helps keep the size on the chosen axis. When flex-direction is row it will keep width as 60 but when flex-direction is column it will keep height as 60px.

flex-basis: autosets width or height explicitly set by the flex item if any else covers as much space as the space required by the flex item
